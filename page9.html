# API

框架提供丰富的微信原生API，可以方便的调起微信提供的能力，如获取用户信息，本地存储，支付功能等。

# API说明

说明

  • wx.on 开头的 API 是监听某个事件发生的API接口，接受一个 CALLBACK回调函数作为参数。当该事件触发时，会调用 CALLBACK回调函数。

  • 如未特殊约定，其他 API 接口都接受一个 OBJECT 作为参数。

  • OBJECT中可以指定success, fail, complete来接收接口调用结果。
